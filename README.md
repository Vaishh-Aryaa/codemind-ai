# CodeMind AI

CodeMind AI is a modern AI-powered code summarizer web app that explains code in simple English using AI.

The app automatically detects programming languages and provides structured explanations for pasted code snippets.

---

## Features

- AI-powered code summarization
- Automatic programming language detection
- Markdown-rendered AI responses
- Dark / Light mode
- Copy AI response button
- Auto-expanding textarea
- Smooth chat experience
- Loading animations
- Responsive modern design

---

## Tech Stack

- HTML
- CSS
- JavaScript
- Vite
- OpenRouter API
- Marked.js
- Font Awesome

---

## Installation

```bash
git clone https://github.com/yourusername/codemind-ai.git
```

```bash
cd codemind-ai
```

```bash
npm install
```

---

## Environment Variables

Create a `.env` file and add:

```env
VITE_OPENROUTER_API_KEY=your_api_key
```

---

## Run Locally

```bash
npm run dev
```
